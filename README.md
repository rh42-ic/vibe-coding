# Vibe Coding Playground

欢迎来到我的 Vibe Coding Playground！

这是一个用于探索和验证 AI 编程工作流的实验室。所有的工具和配置均围绕 **OpenCode** 格式进行构建和优化。

## Skills (技能)

- **Verible Skill**: 为 SystemVerilog/Verilog 提供 Lint、格式化及语法检查的完整工具链支持。

## Subagents (子代理)

`subagents` 目录包含了采用标准 **OpenCode** 格式的自制子代理定义文件。

- **Minion Subagent**: 专为执行简单安全任务设计的子代理，强调“及时放弃并上报”。
